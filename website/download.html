Desktop

Windows · macOS · Linux

The full application: local detection engine, session ledger, pause and engine controls. The daemon binds to localhost only and makes no network requests except the one-time model download.

Planned release practice

  • Direct downloads — a file, a SHA-256 checksum, a signature. No installer that phones home, no account gate, no email capture.
  • The engine ships identical on every tier — see Clause 1.
  • Release notes state what changed in detection — every change to what gets caught or missed is listed, not summarized away.
